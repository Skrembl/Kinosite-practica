Примечание:
Если открываете сайт через 2 способ, то возможно потребуется подождать загрузки в течении 10-60 секунд.

GitHub project: "https://github.com/Skrembl/Skrembl.github.io.git".
Чтобы просматривать, удалять или редактировать отзывы нужно перейти по url("название сайта/admin/login.html").
Способы открытия проекта с рабочей формой отправки отзывов:
	1. Запустить папку проекта через любой редактор кода и в терминале ввести команду php -S localhost:8000.
	2. Открыть публичный сайт по адресу "https://akim-tv.onrender.com".
	3. Чтобы вручную не запускать локальный хост, можно два раза кликнуть по файлу start_server.bat и сайт откроется автоматически.
